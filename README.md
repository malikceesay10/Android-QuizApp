## Projektbeschreibung
Diese Android-App ist ein interaktives Quiz-System mit integrierter Benutzerverwaltung über Firebase Authentication.
Der Nutzer kann sich registrieren, einloggen und zwischen zwei Hauptmodi wählen:
1. **Quiz-Modus**: Fragen beantworten und Punkte sammeln (5-Fragen-Logik).
2. **Lern-Modus**: Fragen einer Kategorie ohne Zeitdruck einsehen.
